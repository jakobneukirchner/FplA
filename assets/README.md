# Assets

Lege hier deine Branding-Dateien ab:

- `logo.svg` oder `logo.png` – wird in der Kopfzeile angezeigt
- `favicon.ico` – Browser-Tab-Icon
- `branding.css` – überschreibt CSS-Variablen (Farben, Schriften)

## branding.css Beispiel

```css
:root {
  --primary: #003d82;       /* Hauptfarbe (Header, Buttons) */
  --accent:  #f39200;       /* Akzentfarbe */
  --logo-display: block;    /* 'none' um Logo auszublenden */
}
```
